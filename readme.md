# PJ Outdoors Container Inventory System

Container inventory management system for tracking products across 8 storage containers.

## Quick Start

```bash
npm install
npm start
```

Open: http://localhost:3000

## Deploy to Railway

1. Push to GitHub
2. Connect to Railway
3. Deploy automatically

## Features

- 8 container tracking
- Location: Container, Side (A-N), Shelf
- Search products
- Low stock alerts
- Mobile responsive
